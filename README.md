# Course Goals Tracker

First TypeScript + React project - learning TypeScript fundamentals through a practical goal tracking app.

## Live Demo
[View Live App](https://oizzoo.github.io/first-TS-mini-project/)

## ✨ Features
- Add and delete course goals
- Track goal descriptions
- Smart warning system (alerts when too many goals)
- TypeScript type safety throughout

## Tech Stack
- **React** - UI library
- **TypeScript** - Type safety
- **Vite** - Build tool
- **CSS** - Styling

## Learning Goals
This project helped me learn:
- TypeScript with React (useState, useRef)
- Type-safe props and event handlers
- Discriminated unions (InfoBox component)
- Form handling in TypeScript
- Conditional rendering

## Installation
```bash
npm install
npm run dev
```

## Deployment
```bash
npm run build
npm run deploy
```
